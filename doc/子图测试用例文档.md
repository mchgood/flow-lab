# 子图解析测试用例文档（解析器模块）

## 1. 范围与位置
子图测试归属**解析器模块**，集中在 `FlowParserServiceTest` 中，用于验证 `subgraph ... end` 语法与主图连接的解析结果。

## 2. 用例清单（对应测试方法）

1) `testParseSimpleSubgraph`
- 场景：单个子图 + 主图连边
- 关注点：子图内部边、主图到子图、子图到主图

2) `testParseSubgraphWithDifferentNodeShapes`
- 场景：子图内包含菱形/矩形/圆形
- 关注点：形状识别与边标签保留

3) `testParseMultipleSubgraphs`
- 场景：多个并列子图
- 关注点：各子图内部边 + 子图之间连接

4) `testParseSubgraphWithComplexFlow`
- 场景：子图内循环与决策分支
- 关注点：循环边与分支边完整解析

5) `testParseSubgraphWithOnlyNodes`
- 场景：子图仅定义节点，无内部边
- 关注点：主图连边正常解析

6) `testParseSubgraphWithParallelBranches`
- 场景：子图并行分支汇聚
- 关注点：汇聚节点入边与主图出边

7) `testParseSubgraphWithRoundRectangleNodes`
- 场景：子图内圆角矩形节点
- 关注点：`round_rectangle` 形状识别

8) `testParseEmptySubgraph`
- 场景：空子图
- 关注点：空子图不影响整体解析

## 3. 运行方式
```bash
# 子图相关测试
./mvnw test -Dtest=FlowParserServiceTest#testParse*Subgraph*
```
